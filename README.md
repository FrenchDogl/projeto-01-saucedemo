# Projeto 01 - Testes Manuais Sauce Demo

## Objetivo 
Realizar testes manuais na funcionalidade de login do Suce Demo. 

## Escopo
- Tela de Login
- Validação de credenciais
- Validação de mensagens de erro
- Testes de usabilidade

## Técnicas Utilizadas
- Testes positivos
- Testes negativos
- Testes de borda

## Resultados 
- 30 cenários criados
- 30 cenários executados
- 26 evidências coletadas
- Nenhum defeito encontrado

## Aprendizados 
- Construção de cenários de teste
- Pensamento baseado em riscos
- Documentação de evidências

- Verificação de itens

- Relatório de bugs mais profissional e empresarial
